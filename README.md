#  我的博客

基于 VitePress 构建的个人技术博客。

##  快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```

##  使用

在 `docs/notes/` 目录下创建文件夹和 Markdown 文件，侧边栏会自动生成。

##  部署到 Vercel

1. 推送代码到 GitHub
2. 访问 [vercel.com](https://vercel.com)
3. 导入你的 GitHub 仓库
4. 点击部署
